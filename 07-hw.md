
### 1. Разложение $x^7-1$ над полем $\mathbb{F}_2$

Поле $\mathbb{F}_2$ имеет характеристику $2$, поэтому $-1 = 1$ и вычитание совпадает со сложением. Следовательно:

$$
x^7 - 1 = x^7 + 1.
$$

Проверим наличие линейных множителей. Подстановка $x=1$ даёт:

$$
1^7 + 1 = 1 + 1 = 0 \pmod 2,
$$
значит, $x=1$ является корнем, а $(x+1)$ делит многочлен. Выполним деление (или воспользуемся формулой суммы геометрической прогрессии в $\mathbb{F}_2$):
$$
x^7 + 1 = (x + 1)(x^6 + x^5 + x^4 + x^3 + x^2 + x + 1).
$$

Обозначим $f(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$. Он не имеет корней в $\mathbb{F}_2$ ($f(0)=1$, $f(1)=1$), следовательно, не содержит линейных множителей. Поскольку его степень равна $6$, он либо неприводим, либо распадается на произведение двух неприводимых многочленов степени $3$. Над $\mathbb{F}_2$ известно ровно два неприводимых кубических многочлена:

$$
p_1(x) = x^3 + x + 1, \quad p_2(x) = x^3 + x^2 + 1.
$$

Проверим их произведение:

$$
\begin{aligned}
p_1(x)p_2(x) &= (x^3 + x + 1)(x^3 + x^2 + 1) \\
&= x^6 + x^5 + x^3 + x^4 + x^3 + x + x^3 + x^2 + 1 \\
&= x^6 + x^5 + x^4 + 3x^3 + x^2 + x + 1 \\
&= x^6 + x^5 + x^4 + x^3 + x^2 + x + 1 \pmod 2.
\end{aligned}
$$

Совпадение подтверждено. Таким образом, каноническое разложение на неприводимые множители над $\mathbb{F}_2$ имеет вид:

$$
\boxed{x^7 - 1 = (x + 1)(x^3 + x + 1)(x^3 + x^2 + 1)}.
$$

### 2. Построение циклических кодов длины $n=7$

Циклический код длины $n$ над $\mathbb{F}_2$ однозначно задаётся своим **порождающим многочленом** $g(x)$, который должен удовлетворять условию $g(x) \mid (x^n - 1)$. Параметры кода обозначаются как $[n, k, d]$, где:
- $n = 7$ длина кода,
- $k = n - \deg g(x)$ размерность (число информационных символов),
- $d$ минимальное хеммингово расстояние (минимальный вес ненулевого кодового слова).

Многочлен $x^7-1$ распадается на три попарно различных неприводимых множителя. Общее число его делителей равно $2^3 = 8$. Исключая тривиальные случаи $g(x)=1$ (код без избыточности, $k=7$) и $g(x)=x^7-1$ (нулевой код, $k=0$), остаётся ровно **6 нетривиальных порождающих многочленов**. Рассмотрим каждый из них.

#### 1. $g_1(x) = x + 1$
- $\deg g_1 = 1 \Rightarrow k = 7 - 1 = 6$. Параметры: $[7, 6]$.
- Код состоит из всех векторов чётного веса. Наименьший ненулевой вес равен $2$ (любой вектор с двумя единицами).
- **Минимальное расстояние:** $d = 2$.
- **Тип:** Код с проверкой на чётность.

#### 2. $g_2(x) = x^3 + x + 1$
- $\deg g_2 = 3 \Rightarrow k = 7 - 3 = 4$. Параметры: $[7, 4]$.
- Многочлен $g_2(x)$ примитивен: его корень порождает мультипликативную группу поля расширения $\mathbb{F}_{2^3}$. По свойствам теории кодирования, такой многочлен порождает классический двоичный код Хэмминга.
- Для кода Хэмминга с $r=3$ проверочными символами минимальное расстояние всегда равно $3$.
- **Минимальное расстояние:** $d = 3$.
- **Тип:** Код Хэмминга $[7,4,3]$.

#### 3. $g_3(x) = x^3 + x^2 + 1$
- $\deg g_3 = 3 \Rightarrow k = 4$. Параметры: $[7, 4]$.
- Данный многочлен является обратным к $g_2(x)$ (коэффициенты записаны в обратном порядке). Он также неприводим и примитивен.
- Порождает код, эквивалентный коду Хэмминга (отличается лишь перестановкой координат).
- **Минимальное расстояние:** $d = 3$.
- **Тип:** Код Хэмминга $[7,4,3]$ (эквивалентное представление).

#### 4. $g_4(x) = (x + 1)(x^3 + x + 1) = x^4 + x^3 + x^2 + 1$
- $\deg g_4 = 4 \Rightarrow k = 7 - 4 = 3$. Параметры: $[7, 3]$.
- Проверочный многочлен этого кода равен $h(x) = (x^7-1)/g_4(x) = x^3 + x^2 + 1 = g_3(x)$. Следовательно, данный код является двойственным к коду Хэмминга, порождённому $g_3(x)$.
- Двойственный код к $[7,4,3]$ Хэмминга известен как **симплексный код**. Все его $2^3-1=7$ ненулевых кодовых слов имеют одинаковый хеммингов вес $2^{m-1} = 2^{3-1} = 4$.
- **Минимальное расстояние:** $d = 4$.
- **Тип:** Симплексный код (двойственный к коду Хэмминга) $[7,3,4]$.

#### 5. $g_5(x) = (x + 1)(x^3 + x^2 + 1) = x^4 + x^2 + x + 1$
- $\deg g_5 = 4 \Rightarrow k = 3$. Параметры: $[7, 3]$.
- Аналогично предыдущему случаю, проверочный многочлен $h(x) = x^3 + x + 1 = g_2(x)$. Код двойственен коду Хэмминга, порождённому $g_2(x)$.
- Обладает теми же метрическими свойствами: постоянный вес ненулевых слов равен $4$.
- **Минимальное расстояние:** $d = 4$.
- **Тип:** Симплексный код $[7,3,4]$.

#### 6. $g_6(x) = (x^3 + x + 1)(x^3 + x^2 + 1) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$
- $\deg g_6 = 6 \Rightarrow k = 7 - 6 = 1$. Параметры: $[7, 1]$.
- Размерность $k=1$ означает, что информационное слово состоит из одного бита. При кодировании бит повторяется $7$ раз: $0 \mapsto (0,\dots,0)$, $1 \mapsto (1,\dots,1)$.
- Ненулевое кодовое слово имеет вес $7$.
- **Минимальное расстояние:** $d = 7$.
- **Тип:** Код с повторением $[7,1,7]$.

---

### 3. Ответ

Разложение многочлена над $\mathbb{F}_2$:
$$
x^7 - 1 = (x + 1)(x^3 + x + 1)(x^3 + x^2 + 1).
$$

Существует ровно шесть нетривиальных циклических кодов длины $7$ над $\mathbb{F}_2$. Их параметры и классификация:

1. $g(x) = x + 1$ $\;\Rightarrow\;$ $[7, 6, 2]$ код с проверкой на чётность.
2. $g(x) = x^3 + x + 1$ $\;\Rightarrow\;$ $[7, 4, 3]$ код Хэмминга.
3. $g(x) = x^3 + x^2 + 1$ $\;\Rightarrow\;$ $[7, 4, 3]$ код Хэмминга (эквивалентный).
4. $g(x) = x^4 + x^3 + x^2 + 1$ $\;\Rightarrow\;$ $[7, 3, 4]$ симплексный код.
5. $g(x) = x^4 + x^2 + x + 1$ $\;\Rightarrow\;$ $[7, 3, 4]$ симплексный код.
6. $g(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$ $\;\Rightarrow\;$ $[7, 1, 7]$ код с повторением.

Все приведённые вычисления опираются на теорию циклических кодов, свойства неприводимых многочленов над конечными полями и стандартные метрические характеристики классических кодов. Решения обоих вариантов математически верны и полностью согласуются между собой; данное оформление объединяет их в строгую последовательную форму.
